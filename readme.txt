== Description ==

O WP AI Comment Responder é um plugin que utiliza a API da OpenAI para responder automaticamente a comentários no seu site WordPress. Ele usa o modelo GPT-4o-mini para gerar respostas inteligentes e relevantes, economizando tempo e esforço.

== Installation ==

1. Faça o upload dos arquivos do plugin para o diretório `/wp-content/plugins/wp-ai-comment-responder`, ou instale o plugin diretamente através da tela de plugins do WordPress.
2. Ative o plugin através da tela 'Plugins' no WordPress.
3. Configure suas chaves de API e outras opções na página de configurações do plugin no menu Configurações -> WP Ai Comment Responder.
4. É recomendado o uso do plugin Akismet Anti-spam para filtrar comentários de spam e economizar créditos

== Frequently Asked Questions ==

= Como configuro minhas chaves de API? =

Você pode configurar suas chaves de API na página de configurações do plugin, acessível através do menu de administração do WordPress.

= O que acontece se minha licença expirar? =

Se sua licença expirar, o plugin exibirá uma notificação na área de administração do WordPress até que a licença seja renovada.
