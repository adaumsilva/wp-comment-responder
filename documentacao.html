Instalação

  1. Baixar o Plugin: Baixe o arquivo ZIP do plugin.
  2. Instalar o Plugin:
    • Acesse o painel de administração do WordPress.
    • Navegue até Plugins > Adicionar Novo.
    • Clique em Enviar Plugin e selecione o arquivo ZIP baixado.
    • Clique em Instalar Agora.
  3. Ativar o Plugin: Após a instalação, clique em Ativar Plugin.

Ativação da Licença

Para ativar a licença do plugin, siga os passos abaixo:

  1. Acesse o painel de administração do WordPress.
  2. Navegue até Configurações > WP AI Comment Responder.
  3. Insira a chave da licença no campo Chave da Licença.
  4. Clique em Ativar Licença.

Configuração

Para configurar o plugin, siga os passos abaixo:

  1. Acesse o painel de administração do WordPress.
  2. Navegue até Configurações > WP AI Comment Responder.
  3. Preencha os seguintes campos:
    • Chave da API OpenAI: Insira sua chave da API OpenAI.
  4. Clique em Salvar Alterações.

Opções de Assistente

O plugin oferece várias opções de assistente que podem ser configuradas:

  • Usar Assistente: Marque esta opção para usar a API de assistants da OpenAI. Podendo subir arquivos e treinar seu assistente para melhor qualidade nas respostas.
  • ID do Assistente: Insira o ID do assistente.
  • Nome do Assistente: Crie um nome que será usado como nome de quem irá responder os comentários.
  • Habilitar Pesquisa de Arquivos: Caso esteja usando pesquisa de arquivos no seu assistente, marque esta opção.
  • Habilitar Interpretador de Código: Caso esteja usando o interpretador de código no seu assistente, marque essa opção.

Essas opções podem ser configuradas na página de configurações do plugin.
